# Mobile App Roadmap

## Phase 1

- Set up `apps/mobile` with Expo Router and TypeScript
- Reuse Django auth and branding APIs
- Load org-admin-driven primary color theme
- Build shared mobile shell and product switcher

## Phase 2

- Add auth flow, secure token/session handling, and subscription-aware access
- Build shared mobile components for forms, lists, cards, filters, and empty states
- Convert desktop table-heavy screens into card stacks and detail views

## Phase 3

- Roll out `Work Suite`, `Business Autopilot`, `Storage`, and `AI Chatbot` modules
- Add push notifications, uploads, caching, and better analytics views
- Prepare OTA-safe updates for JS-only changes

## Phase 4

- Real-device QA
- Android and iOS release preparation
- App Store and Play Store publishing
